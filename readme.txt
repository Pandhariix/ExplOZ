Salut tout le monde, ceci est un code de test pour apprendre a utiliser gitHub.

L'idée pour modifier un code, c'est :

- ne pas toucher à la branche master, créer une branche à soi (en la nommant de préférence suivant votre pseudo)
- modifier le code DANS SA BRANCHE
- Compare and create pull request, en laissant un petit commentaire pour expliquer ce que vous avez fait.

De la, deux posibilité : soit vous pouvez fusionner votre code avec le code master, dans ce cas tout est merveilleux, vous fusionnez et vous supprimez votre branche
soit votre request est rejetée, et la il faut gerer les conflits de version.

C'est un peu complexe mais indispensable, on en reparlera en long en large et en travers
